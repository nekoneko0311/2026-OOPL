# Game Introduction

這是一款經典的固定數值 RPG (角色扮演) 遊戲。玩家扮演勇者在塔中探索，收集資源（如鑰匙、血瓶、寶石），擊敗怪物，最終打敗魔王。遊戲的核心在於數值的計算與資源的分配，玩家必須精打細算自己的血量、攻擊力與防禦力，尋找最優路線來通關。
目前實作重點將放在遊戲的「上層邏輯」與玩法呈現。
為了降低通關難度，應該會將低一些難度與怪物數量。

實作功能：

1. **開始遊戲**：簡單的開始按鈕
1. **地圖與角色移動**：讀取網格化的地圖資料，實作勇者在二維網格上的四方向移動。
2. **物件互動與碰撞**：遇到牆壁無法通行；遇到門需要消耗對應顏色的鑰匙才能開啟並通過。
3. **物品收集**：在地圖上撿取各類資源（紅藍寶石提升攻防、血瓶恢復生命、鑰匙開啟門），並更新角色狀態。
4. **戰鬥系統**：遇到地圖上的怪物即觸發戰鬥。採用固定數值回合結算（比較雙方攻防計算損血），若角色血量足以擊殺怪物則扣除相應血量並消滅怪物，獲得金幣/經驗值。
5. **數值狀態 UI 顯示**：在遊戲視窗邊緣或側邊，即時顯示勇者的生命值、攻擊力、防禦力、金幣、經驗與各色鑰匙數量。
6. **NPC 與商店系統**：實作與特定 NPC 互動開啟選單，允許玩家消耗金幣或經驗值來提升自身能力。

# Game Introduction

你說的對，但明日方舟是一款由鷹角自主研發的二次元塔防遊戲。我們期望重建其中之一的關卡在地圖中會有敵人刷新點、防守點、可放幹員點。我們需要部屬幹員到地圖上，若太多敵人成功進入防守點則失敗，反之則成功。
我猜我們只能搓少數幾個屬性的幹員且少數敵人且只有一關。

# Game Introduction
在遊戲中，玩家將操控一名守護者英雄進入下水道，利用走位、普通攻擊與傭兵協作，消滅不斷湧出的敵方怪物。透過擊敗敵人獲取經驗值升級後，玩家可以選擇獲得額外增益屬性，或召喚傭兵協助作戰。
玩家可攜帶各種武器進入戰鬥，不同武器與傭兵皆具備相異的能力與技能。當滿足特定的升級圖譜時，可以將三名傭兵合成為強大的「特殊傭兵」；若達成特定條件，玩家還能獲得額外的「羈絆技能」。
敵方單位種類繁多（如近戰小兵、遠程射手、自爆怪等），分別具有不同的追蹤行為與攻擊模式。每個關卡由三種固定區域組成，雖然區域類別固定，但內部的地形為隨機生成。
英雄初始擁有固定的生命值，但根據武器類型有不同的生命數量。玩家成功擊敗所有敵人即通關；反之，若英雄死亡或場上的怪物數量達到上限，則遊戲結束。

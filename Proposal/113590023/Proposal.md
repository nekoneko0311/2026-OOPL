# Game Introduction

**遊戲簡介** <br>
本遊戲是一款經典的 2D 橫向卷軸平台跳躍遊戲，旨在重現馬力歐的核心玩法。玩家將扮演主角馬力歐，在充滿各種地形與敵人的蘑菇王國中冒險，收集金幣、避開陷阱，並擊敗敵人。

**期望完成內容** <br>
專案預計實作完整的角色物理系統（包含重力、跳躍、衝刺）、地圖動態加載系統、碰撞偵測，以及基礎的 UI 介面（分數、生命數、倒數計時）。

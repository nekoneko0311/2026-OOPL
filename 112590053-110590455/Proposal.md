# Game Introduction

本專題將以經典塔防遊戲「植物大戰殭屍」為原型進行遊戲複刻與實作。玩家需要在草地上種植不同功能的植物來抵禦不斷進攻的殭屍，透過策略性配置植物位置來保護自己的基地。

遊戲的基本玩法為：玩家利用陽光（Sun）資源購買植物，例如向日葵可以生產陽光、豌豆射手可以攻擊殭屍、堅果牆可以阻擋殭屍前進。殭屍會從地圖右側出現並向左前進，若殭屍成功突破防線進入玩家基地，遊戲即判定失敗。
